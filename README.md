# Análisis de rendimiento de tiendas — Alura Store

## Descripción del proyecto

Este proyecto consiste en un análisis exploratorio de datos con el objetivo de evaluar el desempeño de las cuatro tiendas de la cadena **Alura Store**.

A partir de los datos de ventas, calificaciones de clientes y costos de envío, se busca identificar cuál tienda presenta el menor rendimiento general para recomendar cuál debería vender el Sr. Juan con el fin de iniciar un nuevo emprendimiento.

El análisis se realizó utilizando herramientas de **Data Science en Python**, generando visualizaciones que permiten identificar patrones, comparar métricas y fundamentar la decisión final.

---

# Objetivos del análisis

El proyecto analiza distintos indicadores clave del negocio:

- **Ingresos totales por tienda**
- **Categorías de productos más vendidas**
- **Calificaciones promedio de los clientes**
- **Productos más vendidos y menos vendidos**
- **Costo promedio de envío por tienda**

A partir de estas métricas se realiza una comparación entre las tiendas para identificar fortalezas y debilidades en su desempeño.

---

# Tecnologías utilizadas

- Python  
- Pandas — manipulación y análisis de datos  
- Matplotlib — visualización de datos  
- Google Colab — entorno de desarrollo

# Análisis realizado

Durante el proyecto se llevaron a cabo los siguientes análisis:

##Ingresos totales por tienda

Se calculó el ingreso total sumando el precio de todos los productos vendidos en cada tienda.

Este análisis permite identificar qué tienda genera mayor volumen de ventas dentro de la cadena.

---

## Categorías de productos más vendidas

Se agruparon los datos por categoría de producto para determinar cuáles son las categorías con mayor número de ventas en cada tienda.

Este análisis ayuda a comprender los patrones de consumo de los clientes.

---

## Calificaciones promedio de clientes

Se calcularon los promedios de las evaluaciones realizadas por los clientes para cada tienda.

Las calificaciones permiten medir el nivel de satisfacción del cliente con la experiencia de compra.

---

## Productos más y menos vendidos

Se identificaron los productos con mayor y menor volumen de ventas en cada tienda.

Esto permite reconocer qué productos impulsan las ventas y cuáles presentan baja rotación.

---

## Costos promedio de envío

Se calculó el costo promedio de envío para cada tienda.

Dado que el costo de envío es asumido por el cliente, este factor puede influir en la decisión de compra y en la percepción del servicio.

---

# Visualizaciones

Para facilitar la interpretación de los datos, se generaron distintos tipos de gráficos:

- gráficos de barras  
- gráficos comparativos entre tiendas  
- visualizaciones de tendencias  
- gráficos de categorías

Las visualizaciones permiten identificar rápidamente diferencias entre las tiendas y respaldar el análisis realizado.

---

# Conclusión

A partir del análisis de los distintos indicadores, se identificó que **la Tienda 4 presenta el menor nivel de ingresos dentro de la cadena**, lo que indica un desempeño comercial inferior en comparación con las demás tiendas.

Si bien otras tiendas presentan oportunidades de mejora en indicadores como satisfacción del cliente o costos de envío, la Tienda 4 es la que muestra el menor impacto en los resultados generales del negocio.

Por este motivo, se recomienda considerar **la venta de la Tienda 4**, permitiendo concentrar recursos en las unidades con mejor desempeño o destinarlos al nuevo emprendimiento del Sr. Juan.
